# Plantillas de Graficos Para Subir

Estas piezas son ejemplos de como deben generarse los archivos graficos para CharroPro.

Regla principal:

- La imagen trae estilo, textura, marco, luz y decoracion.
- La app pone encima nombres, puntos, filas, oportunidades, cronometro y datos variables.

## Archivos recomendados

| Archivo | Tamano sugerido | Uso |
| --- | ---: | --- |
| `barra-marcador-bg.svg` | 860 x 96 | Fondo de cada equipo en marcador |
| `ranking-bg.svg` | 900 x 330 | Fondo/marco de ranking, sin filas dibujadas |
| `turno-bg.svg` | 920 x 150 | Fondo del grafico de turno |
| `cronometro-bg.svg` | 390 x 150 | Fondo del cronometro |
| `categoria-bg.svg` | 760 x 170 | Fondo para categoria |
| `coleadero-bg.svg` | 760 x 430 | Fondo para coleadero |
| `foto-charro-frame.svg` | 360 x 420 | Marco para foto de charro |
| `marca-agua-bg.svg` | 900 x 420 | Marca de agua o fondo tenue |
| `guia-zonas.svg` | 1280 x 720 | Guia de zonas seguras, no subir como grafico final |

## Como subirlo al host

Ejemplo de ruta:

```text
https://orgullocharro.com/charropro/assets/themes/mi-tema/barra-marcador-bg.svg
```

Para PNG o WebP usa los mismos nombres:

```text
barra-marcador-bg.png
ranking-bg.webp
turno-bg.png
```

## Importante

No pongas nombres de equipos, puntos, numeros de ranking ni lineas de cada fila dentro de la imagen. Eso lo genera la app para que el grafico crezca o se reduzca segun los datos.
